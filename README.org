*** Sticky =MOU= layer
After holding the corresponding layer key and pressing any key defined in =MOU=, the layer remains active even after releasing the key.
The layer automatically turns off when a =KC_TRNS= or =KC_NO= key is pressed.
These layers are mostly used for navigation keys, which are often pressed in sequences. Keeping the layer sticky prevents strain from holding the layer key for extended periods â€” you only need to hold it for the first keypress.
